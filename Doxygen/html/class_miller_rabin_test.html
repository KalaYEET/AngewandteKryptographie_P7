Funktion vom Datentyp bool, welche einen Primzahltest ausführt im Fall einer Primzahl false ausgibt
und true im Fall das Zahl a ein solider Beweis dafür ist, dass n eine zusammengesetzte Zahl ist.

Fall 1: a^n−1 !≡ 1 (mod n). Also kann n wegen dem Satz von Fermat keine Primzahl sein.
Fall 2: a^n−1 ≡ 1 (mod n). Dann existiert ein i ∈ {1, . . . r}, so dass
a^u*2^i ≡ 1 (mod n) und a^u*2^i-1 !≡ ±1 (mod n). Also kann n keine
Primzahl sein, da es eine von ±1 verschiedene Quadratwurzel von 1
modulo n gibt.

Parameters
areferenz zur Zufallszahl
nreferenz zur auf Prim getesten Zahl
Returns
true oder false: a solider Beweis für n , dass n eine zusammengesetzte Zahl ist oder Primzahl
